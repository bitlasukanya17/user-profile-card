# User Profile Card Generator

A beautiful Flask web application that generates stylish user profile cards using user input.

---

## Features

- User profile form
- Dynamic profile card generation
- Attractive modern UI
- Responsive design
- Flask backend with POST request handling

---

## Technologies Used

- Python
- Flask
- HTML5
- CSS3
